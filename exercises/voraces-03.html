03 — Generalización del Algoritmo Voraz

Dado un problema, identifica los cuatro componentes del esquema voraz general: C (candidatos), S (solución parcial), f (función objetivo) y R (restricciones). Pulsa el botón correcto en cada tarjeta.

Ronda 1 A la primera: 0 / 0
—

C Candidatos — elementos que pueden incorporarse a la solución ✓
S Solución — subconjunto construido progresivamente, empieza vacío ✓
f Objetivo — qué queremos maximizar o minimizar ✓
R Restricciones — condiciones que definen cuándo una solución es válida ✓

// C: candidatos S: solución f: objetivo R: restricciones
function greedy(C, R)
  S = ∅
  while C ≠ ∅ and not complete(S) do
    c = select_best_feasible(S, C, R)
    if c ≠ NULL then
      S = S ∪ {c} ; C = C − {c}
    else break
  end_while
  if complete(S) then return S else return ∅
